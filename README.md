# LuxeTex Industries Website

This project is a professional, 3D-styled website for the Textile Industry.

## Project Structure

- **base.html**: Main layout template (reference).
- **index.html**: Home page with Hero 3D animations.
- **about.html**: Company history and mission.
- **services.html**: Overview of services.
- **service-1.html**: Yarn Manufacturing details.
- **service-2.html**: Fabric Dyeing details.
- **service-3.html**: Textile Printing details.
- **contact.html**: 3D Contact form.
- **privacy.html**: Privacy Policy.
- **static/**: Contains CSS, JS, and Images (external links).

## Features

- **3D Design**: Cards and elements react to mouse movement (tilt effect).
- **Responsive**: Built with Bootstrap 5.
- **Custom Styling**: `style.css` contains the "Deep Navy & Gold" theme.

## How to Run

Simply open `index.html` in any modern web browser.
